Rules

· There is only one valid solution to each Sudoku puzzle. The only way the puzzle can be considered solved correctly is when all 81 boxes contain numbers and the other Sudoku rules have been followed. · When you start a game of Sudoku, some blocks will be pre-filled for you. You cannot change these numbers in the course of the game. · Each column must contain all of the numbers 1 through 9 and no two numbers in the same column of a Sudoku puzzle can be the same. · Each row must contain all of the numbers 1 through 9 and no two numbers in the same row of a Sudoku puzzle can be the same. · Each block must contain all of the numbers 1 through 9 and no two numbers in the same block of a Sudoku puzzle can be the same..

Source code & implementation

I developed a Ruby on Rails basic application of the sudoku game using the following:
Fedora 21 x_64 OS
Ruby version 2.0.0
gem 'rails', '4.2.1'
gem 'sass-rails', '~> 5.0' As CSS managers
gem 'uglifier', '>= 1.3.0' JS compressor
gem 'coffee-rails', '~> 4.1.0' coffee assets and views
gem 'jquery-rails' Jquery
gem 'turbolinks' to faster links
gem 'sudoku_maker' This gem provides an array that was used to generate Sudokus, and locating this array at the solution, the user can see the puzzle solved.
gem 'sqlite3' To manage DB locally, at hosted app I used Postgres because Herokuapp only supports Postgres DB.
gem 'spring' for keep the app running in background
gem 'pg' Postgres DB management
000webhost to host this page
Heroku to host the DB and the entire Ruby on rails Sudoku app.
